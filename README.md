# Event Photography Studio Website UI Design

## Project Overview
This project contains the complete UI design for a photography studio website. 
The goal of this design is to create a premium, conversion-focused experience 
for users looking to book photography services.

## Objective
To design a clean, modern, and user-friendly website interface that:
- Showcases photography work professionally
- Builds trust through social proof
- Encourages users to inquire or book services

## Target Users
- Engaged couples planning weddings
- Families looking for event photography
- Brands seeking professional photoshoots
- Individuals looking for model or portfolio shoots

## Pages Included
1. Homepage  
2. Service Detail Page  
3. Contact / Lead Generation Page  

## Design Approach
- Strong visual hierarchy
- Luxury and elegant aesthetics
- Clear call-to-action buttons
- Minimal navigation for better focus
- Emotion-driven imagery

## Conversion Strategy
- Hero section with clear CTA
- Testimonials for trust building
- Service breakdown to reduce confusion
- Simple contact form to reduce drop-off
- Social proof to increase credibility
